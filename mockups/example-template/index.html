Шаблон-пример

Это стартовый шаблон для создания новых макетов

О шаблоне

Этот шаблон предоставляет базовую структуру для ваших макетов. Он включает:

  • Чистую HTML5 структуру
  • Базовые CSS стили в css/style.css
  • Опциональные общие утилиты из shared/css/common.css
  • Папку assets/ для изображений и других ресурсов

Начало работы

Чтобы создать новый макет на основе этого шаблона:

  1. Скопируйте эту папку в новое место в директории mockups/
  2. Переименуйте папку в соответствии с названием вашего макета
  3. Отредактируйте этот HTML файл с вашим контентом
  4. Настройте стили в css/style.css
  5. Добавьте любые изображения или ресурсы в папку assets/
  6. Обновите страницу каталога, добавив ссылку на ваш новый макет
