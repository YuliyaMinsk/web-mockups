Как добавить новый макет

  1. Скопируйте папку mockups/example-template/ в mockups/{имя-вашего-макета}/
  2. Отредактируйте HTML и CSS файлы в новой папке
  3. Добавьте новую карточку на эту страницу с описанием вашего макета
  4. Закоммитьте и запуште изменения
